- (vim-adventure)[https://vim-adventures.com/]

## i

일반모드에서 INSERT MODE로 진입하는 명령어

## esc

일반모드로 빠져나온다.

## :

일반모드에서 `:` 입력 시 명령모드로 진입합니다.

명령모드에서 사용할 수 있는 명령어

- :w 저장
- :q 종료
- :q! 저장없이 종료
- :wq! 저장 후 종료
